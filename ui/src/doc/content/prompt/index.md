# 提示词模板管理

系统提示词是智能体的"角色定义与行为准则"，通过结构化指令为大模型设定明确的角色定位、能力边界和交互范式。精心设计的提示词模板能够显著提升智能体在特定领域的表现一致性、专业性及安全性。


## 功能概述

- 支持按分类管理提示词模板
- 提供模板内容编辑器（Markdown 语法）
- 智能体可选择模板并支持"随模板变化"自动更新
- 修改/删除模板时自动触发关联智能体重新注册


## 核心概念

### 模板与自定义提示词

智能体的系统提示词有两种来源：

| 模式 | 说明 | 更新方式 |
|------|------|----------|
| **模板模式** | 选择已有模板，提示词内容来自模板 | 随模板更新自动同步 |
| **自定义模式** | 不使用模板，手动编写提示词内容 | 手动修改 |

### 随模板变化

当智能体关联了提示词模板后，可以开启 **随模板变化** 开关：

- **开启**：提示词内容始终与模板保持一致，模板更新后智能体自动使用最新内容
- **关闭**：选择模板时会将模板内容复制到系统提示词中，之后可自由编辑，不再随模板变化


## 操作指南

### 创建提示词模板

1. 在提示词模板管理页面点击 **添加新提示词模板** 卡片
2. 填写表单：
   - **分类**：选择已有分类或输入新分类名称
   - **名称**：模板名称
   - **描述**：模板用途说明
   - **内容**：使用 Markdown 编辑器编写提示词内容
3. 点击确认提交

### 查看详情

点击卡片右上角菜单 → **查看**，查看模板完整信息，详情弹窗展示以下字段：

| 字段 | 说明 |
|------|------|
| **关联智能体** | 引用此模板的智能体列表 |
| **名称** | 模板名称 |
| **分类** | 模板所属分类 |
| **描述** | 模板用途说明 |
| **提示词内容** | 完整的提示词文本（以代码块形式展示） |

### 编辑模板

点击卡片右上角菜单 → **编辑**，修改模板内容。

:::warning
修改模板后，所有开启"随模板变化"的关联智能体将自动使用新内容并重新注册。
:::

### 启用/禁用

点击卡片右上角菜单 → **启用/禁用**，切换模板的启用状态。

- 禁用的模板不会在智能体配置的模板列表中显示
- 若模板正在被智能体引用，禁用时系统会弹出二次确认提示，显示引用的智能体名称

### 删除模板

点击卡片右上角菜单 → **删除**。

- 若模板正在被智能体引用，系统会弹出二次确认框，显示引用的智能体名称
- 若无引用，弹出常规删除确认框
- 删除操作不可恢复

:::warning
删除被智能体引用的模板时，系统会显示「该提示词模板正在被 [xxx] 智能体引用，删除后可能会影响上述智能体的正常使用」。
:::

### 搜索与筛选

- **分类筛选**：页面顶部分段控制器切换分类，选择后自动刷新列表
- **名称搜索**：输入关键词后回车或点击搜索图标搜索

:::tip
分类列表从数据库中所有模板的 `category` 字段去重获取，进入页面时自动加载。
:::


## 在智能体中使用

在智能体配置的「模型与提示词」步骤中：

1. 选择分类，浏览该分类下的模板列表
2. 选择一个提示词模板（Radio 单选）
3. 配置 **随模板变化** 开关
4. 若关闭随模板变化，可在下方编辑器中自定义提示词内容

## 提示词编写技巧

### 结构化模板

推荐使用以下结构编写提示词：

```markdown
# 角色定义
你是一个专业的客服助手...

## 能力范围
- 解答产品相关问题
- 处理售后投诉...

## 行为准则
- 保持礼貌和专业
- 不确定时诚实回答...

## 输出格式
请按以下格式回复：
1. 问题理解
2. 解决方案
3. 后续建议
```

### 关键要素

- **角色定位**：明确智能体的身份和专业领域
- **能力边界**：清晰界定能做什么、不能做什么
- **行为规范**：设定交互风格和约束条件
- **输出要求**：规定回复格式和质量标准

### 维护建议

- 定期根据实际对话效果优化模板
- 使用分类区分不同场景的模板
- 在描述中注明模板适用场景和注意事项


## 权限说明

| 操作 | 所需角色 |
|------|----------|
| 查看模板列表与详情 | 所有租户成员 |
| 创建、编辑、删除模板 | 租户编辑者、租户管理员、租户所有者 |
| 启用/禁用模板 | 租户编辑者、租户管理员、租户所有者 |


## 常见问题

### 修改模板后智能体没有变化？

- 确认智能体是否开启了「随模板变化」开关
- 如果未开启，智能体使用的是选择模板时复制的快照，不会随模板更新
- 如果已开启，修改模板后智能体会自动重新注册并使用最新内容

### 「随模板变化」开启还是关闭好？

- **开启**：适合团队协作场景，模板维护者更新后所有关联智能体自动生效，无需手动同步
- **关闭**：适合需要针对特定智能体微调提示词的场景，复制模板内容后可自由编辑
- 建议：通用型模板开启，定制化场景关闭

### 提示词模板和自定义提示词有什么区别？

- **模板模式**：从模板库中选择，支持统一管理和批量更新
- **自定义模式**：直接在智能体配置中编写，仅对当前智能体生效
- 自定义模式更适合快速调试，模板模式更适合生产环境的标准化管理

### 如何删除被智能体引用的模板？

- 系统会弹出二次确认框，显示引用的智能体名称
- 可以直接确认删除，但引用方将失去模板关联
- 建议先在相关智能体中解除模板关联，再执行删除

### 分类功能如何使用？

- 创建模板时可选择已有分类或输入新分类名称
- 页面顶部通过分段控制器按分类快速筛选
- 分类列表自动从数据库中已有分类去重生成
- 建议分类示例：客服类、技术类、营销类、通用类

### 提示词编写有什么技巧？

- 明确角色定位：说明智能体的身份和专业领域
- 设定能力边界：清楚界定能做什么、不能做什么
- 规范输出格式：规定回复的结构和质量标准
- 添加示例对话：帮助模型理解期望的交互风格
- 参考文档中的「提示词编写技巧」章节获取更详细的指导
