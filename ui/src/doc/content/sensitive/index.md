# 敏感词管理

敏感词管理模块是企业级智能体应用的核心安全组件，通过系统化的关键词规则配置与实时内容扫描，在智能体交互全流程中构建多层次的内容安全防线。

## 功能概述

- 支持按分类管理敏感词配置，同一分类下配置名称唯一
- 与智能体关联，支持在智能体层面启用/禁用敏感词过滤
- 删除/修改配置时自动触发关联智能体重新注册

## 核心概念

### 分类

分类是敏感词配置的组织维度，用于将不同领域或用途的敏感词进行归类管理。例如：`政治敏感`、`色情暴力`、`商业机密` 等。

- 同一分类下的配置名称必须唯一
- 分类按需自定义，创建配置时可直接新增分类
- 页面顶部支持按分类快速筛选

### 处理动作

| 动作 | 说明 | 额外配置 |
|------|------|----------|
| **警告（WARN）** | 检测到敏感词后发出警告提示，但不阻断对话 | 无 |
| **替换（REPLACE）** | 检测到敏感词后自动替换为指定文本，对话继续 | 需配置替换文本 |

### 敏感词列表

每个敏感词配置包含一组敏感词关键词，支持以下输入方式：

- 输入框输入后按回车添加
- 支持批量输入：使用空格、逗号或顿号分隔多个词
- 自动去重，避免重复添加
- 支持单个删除和一键清空

## 操作指南

### 创建敏感词配置

1. 在敏感词管理页面点击 **添加新敏感词配置** 卡片
2. 填写表单：
   - **分类**：选择已有分类或输入新分类名称
   - **名称**：配置名称（同一分类下唯一）
   - **描述**：配置说明
   - **敏感词列表**：在输入框中输入敏感词，按回车添加
   - **替换文本**：当处理动作为"替换"时，设置替换内容
3. 点击确认提交

### 查看详情

点击卡片右上角菜单 → **查看**，查看配置完整信息，详情弹窗展示以下字段：

| 字段 | 说明 |
|------|------|
| **关联智能体** | 引用此配置的智能体列表 |
| **名称** | 配置名称 |
| **分类** | 所属分类 |
| **描述** | 配置说明 |
| **处理动作** | WARN（警告）或 REPLACE（替换） |
| **替换文本** | 当处理动作为 REPLACE 时显示，展示替换内容 |
| **敏感词列表** | 以标签形式展示所有敏感词关键词 |

### 编辑配置

点击卡片右上角菜单 → **编辑**，修改配置内容。

:::warning
修改敏感词配置后，所有关联该配置的智能体将自动重新注册，以确保新规则立即生效。
:::

### 启用/禁用

点击卡片右上角菜单 → **启用/禁用**，切换配置的启用状态。

- 禁用的敏感词配置不会参与智能体内容过滤
- 若配置正在被智能体引用，禁用时系统会弹出二次确认提示

:::warning
禁用被智能体引用的敏感词配置时，系统会显示「该敏感词正在被 [xxx] 智能体引用，禁用后可能会影响上述智能体的正常使用」。
:::

### 删除配置

点击卡片右上角菜单 → **删除**。

- 若配置正在被智能体引用，系统会弹出二次确认框，显示引用的智能体名称
- 若无引用，弹出常规删除确认框
- 删除操作不可恢复

:::warning
删除被智能体引用的敏感词配置时，确认删除后关联智能体的敏感词过滤将失效。
:::

### 搜索与筛选

- **分类筛选**：页面顶部分段控制器切换分类
- **名称搜索**：输入关键词后回车搜索

## 在智能体中使用

在智能体配置的「工具与能力」步骤中，可以关联敏感词配置：

1. 选择一个敏感词配置（每个智能体只能关联一个）
2. 开启 **启用敏感词过滤** 开关
3. 开启后，智能体将对输入和输出内容进行敏感词过滤

## 最佳实践

### 分类设计建议

- 按风险等级分类：如 `高危`、`中危`、`低危`
- 按业务领域分类：如 `金融合规`、`医疗隐私`、`个人信息`
- 避免分类过多，建议 3~8 个分类

### 处理动作选择

- **警告**：适用于边界内容，提醒用户注意措辞，但不阻断对话流程
- **替换**：适用于明确违规的内容，自动将敏感词替换为指定文本（如 "***"），对话继续

:::tip
选择 REPLACE 动作时，需要在表单中配置替换文本。替换文本会替代检测到的敏感词出现在对话中。
:::

### 敏感词维护

- 定期审查和更新敏感词列表
- 避免过度添加导致误伤正常对话
- 同义词、变体词应一并添加

| 操作 | 所需角色 |
|------|----------|
| 查看敏感词配置列表与详情 | 所有租户成员 |

## 常见问题

### 配置了敏感词为什么不生效？

- 敏感词配置后，还需要在智能体配置中关联该配置并启用敏感词过滤功能
- 进入 **智能体 → 设计 → 配置 → 工具与能力**，选择敏感词配置并勾选「启用敏感词过滤」
- 两个步骤缺一不可

### 替换模式和警告模式如何选择？

- **警告模式**：适合边界词汇，仅提示不阻断，用户体验较好
- **替换模式**：适合明确违规词汇，自动替换为指定文本（如 "***"）
- 建议：先用警告模式观察效果，确认无误后再切换为替换模式

### 如何批量添加敏感词？

- 在输入框中输入多个词，使用空格、逗号或顿号分隔
- 按回车后自动拆分并添加多个词
- 系统自动去重，不会添加重复的敏感词
- 支持单个删除（点击词上的 x）和一键清空

### 同一分类下的名称必须唯一是什么意思？

- 每个分类内的配置名称不能重复
- 不同分类下可以有相同的配置名称
- 例如：「政治敏感」分类下和「商业机密」分类下都可以有名为「基础过滤」的配置

### 删除敏感词配置会影响已关联的智能体吗？

- 删除时系统会弹出二次确认框，显示引用的智能体名称
- 确认删除后，关联智能体的敏感词过滤将失效
- 建议先在智能体配置中解除关联，再执行删除

### 敏感词过滤对智能体性能有影响吗？

- 敏感词过滤在智能体输入输出流程中实时扫描，性能影响较小
- 建议避免在单个配置中添加过多敏感词（超过数千个），可拆分为多个配置
- 每个智能体只能关联一个敏感词配置
